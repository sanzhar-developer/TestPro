TestPro — Система онлайн-тестирования
Полнофункциональное веб-приложение для создания тестов, проведения тестирования и мгновенного вывода результатов. Проект разделен на клиентскую часть (React + Vite) и серверную часть (Node.js + Express).

Основные возможности
Панель администратора: Создание, редактирование и удаление тестов и вопросов прямо в интерфейсе.

Гибкие вопросы: Поддержка одиночного и множественного выбора ответов.

Массовая загрузка: Возможность загрузки вопросов целыми списками через JSON.

Адаптивная верстка: Удобный интерфейс, корректно отображающийся на разных устройствах.

Система авторизации: Разграничение прав доступа для обычных пользователей и администраторов.

Стек технологий
Frontend: React.js, Vite, CSS3 (Flexbox/Grid).

Backend: Node.js, Express.js.

Database: MongoDB (Mongoose).

API Testing: Postman.

Инструкция по запуску
1. Клонирование репозитория

git clone https://github.com/твой-логин/название-репозитория.git
cd название-репозитория

2. Запуск сервера (Backend)

cd server
npm install
npm start

Сервер запустится на порту 5000.

3. Запуск фронтенда (Frontend)

cd ../client
npm install
npm run dev

Приложение будет доступно по адресу http://localhost:5173.

Переменные окружения (.env)
Для корректной работы сервера создайте файл .env в папке server и добавьте:


MONGODB_URI=твоя_ссылка_на_базу_данных
JWT_SECRET=твой_секретный_ключ
PORT=5000
